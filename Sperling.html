Paradigme de Sperling

Test de mémoire sensorielle visuelle

Description de l'expérience ▼

Cette application reproduit le paradigme de Sperling, une expérience classique en psychologie cognitive qui permet d'étudier la mémoire sensorielle visuelle.

Une grille de lettres apparaît brièvement à l’écran. Votre objectif est de mémoriser un maximum de lettres.

Après disparition, vous devez saisir les lettres dont vous vous souvenez.

Cette expérience montre que notre mémoire visuelle est très rapide mais très limitée dans le temps.
