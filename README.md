# 🧠 Paradigme de Sperling - Mémoire Sensorielle

Ce mini-projet est une application web interactive reproduisant le **paradigme de Sperling**, une expérience classique en psychologie cognitive permettant d’étudier la mémoire sensorielle visuelle (mémoire iconique).

---

## 📚 Description

Une grille de lettres est affichée très brièvement à l’écran.
L’utilisateur doit mémoriser un maximum de lettres en un temps très court.

Après disparition :

* L’utilisateur entre les lettres dont il se souvient
* Le système calcule le score et le taux de réussite

---

## ⚙️ Fonctionnalités

* 🔽 Description du test dépliable
* 🔢 Choix du nombre de lignes et colonnes
* ⏱️ Réglage de la durée d’affichage
* 🧪 Expérience interactive
* 📊 Résultats détaillés
* 📈 Moyenne sur plusieurs essais

---

## 🧪 Objectif scientifique

Ce test met en évidence que :

> La mémoire sensorielle visuelle contient beaucoup d’informations, mais pendant une durée très courte (~300 ms).

---

## ▶️ Utilisation

1. Configurer les paramètres (lignes, colonnes, durée)
2. Cliquer sur **Démarrer l’essai**
3. Observer la grille de lettres
4. Entrer les lettres mémorisées
5. Consulter les résultats

---

## 🚀 Lancer le projet

1. Cloner le dépôt :


2. Accéder au dossier :


3. Ouvrir le fichier :

